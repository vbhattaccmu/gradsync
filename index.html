What is gradsync?

gradsync is a complete, working data-parallel training framework that shows how modern GPUs communicate at scale. It demonstrates:

  • How GPUs talk — NVLink/NVSwitch inside a node, NCCL over the network across nodes.
  • How gradients move — in-place AllReduce collectives (GPU-to-GPU, no CPU copies).
  • How real DDP scales — bucketed AllReduce overlapped with backward compute, so communication is "free".
